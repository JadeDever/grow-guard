## 🚀 快速启动

### 1. 环境要求
- Node.js >= 18.0.0
- npm >= 9.0.0

### 2. 启动命令
```bash
# 安装依赖
npm install

# 构建共享包
npm run build:shared

# 启动开发服务器
npm run dev

# 或分别启动
npm run dev:frontend
npm run dev:backend
```

### 3. 访问地址
- 前端: http://localhost:3000
- 后端: http://localhost:3001
- 健康检查: http://localhost:3001/health
